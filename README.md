# GuavaSure — Pet Wellness Website

Static website for [GuavaSure](https://guavasure.com), a pet wellness membership and technology platform based in Bangalore, India.

## Tech Stack

- **HTML5** + **Tailwind CSS** (CDN)
- **Google Fonts**: Noto Serif (headlines) + Be Vietnam Pro (body)
- **Material Symbols Rounded** for icons
- **No build step** — deploy directly to Vercel/Netlify

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero, trust bar, problem stats, how it works, plan card, founder story, blog preview, CTA |
| About | `about.html` | Founder story (Prapanch), origin, approach, Go Digit partnership |
| Plans | `plans.html` | Wellness Plan details, feature grid, cost comparison, FAQ |
| How It Works | `how-it-works.html` | 3-step signup flow, post-signup features |
| Claims | `claims.html` | 4-step claims process, cashless vs reimbursement, coverage grid |
| FAQ | `faq.html` | 17 questions across 5 categories with sidebar nav |
| Contact | `contact.html` | WhatsApp CTA, contact form, email/Instagram |
| Blog | `blog.html` | Article listing with category filters |

## Design System

- **Primary**: `#944521` (terracotta)
- **Secondary**: `#536252` (forest green)
- **Tertiary**: `#426449` (sage)
- **Surface**: `#fff8f2` (warm cream)
- **Typography**: Noto Serif headlines, Be Vietnam Pro body

## Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or connect this repo to Vercel via GitHub for automatic deploys.
